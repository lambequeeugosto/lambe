Sobre este site

  • Este site foi desenvolvido utilizando softwares de código aberto

    Todo este site foi desenvolvido utilizando recursos de software de código aberto e gratuito. Isso porque são mais seguros, mais rápidos e mais acessíveis. Nós acreditamos que a tecnologia deve ser algo acessível a todXs independente de classe social, etnia ou qualquer outra circunstância. Nós utilizamos para o desenvolvimento deste site:

    • Ubuntu (GNU/Linux)

      Ubuntu é um sistema operacional livre baseado no Linux. Ele é um dos maiores e maisinfluentes projetos de código aberto do mundo. Possui funcionalidades incríveis e é continuamente atualizado mantendo sua estabilidade, rapidez e segurança. Escolhemos o Ubuntu como o sistema base para executamos nosso projeto por ele ser mais rápido, estável e bem seguro. Saiba mais sobre o Ubuntu aqui.

    • Brackets

      O Brackets é um editor de textos moderno e de código aberto que entende web design. É desenvolvido pela Adobe em parceria com a Comunidade de Software Livre Brackets. Com ferramentas visuais especializadas e suporte a preprocessadores, o Brackets é um editor de textos moderno que facilita a criação de designs no navegador. Utilizamos o Brackets para escrevermos o código-fonte do nosso site. Saiba mais sobre o Brackets aqui.

    • Gimp

      GIMP é um editor de imagens multi-plataforma. Ele é livre, você pode mudar seu código e redistribuí-lo. Seja você designer gráfico, fotógrafX ou illustrador(a), o GIMP provê ferramentas sofisticadas para desenvolvimento de vários trabalhos. Utilizamos o GIMP para editarmos as imagens e ícones do site. Nós o escolhemos por ser mais rápido. Saiba mais sobre o GIMP aqui.

    • Inkscape

      Inkscape é um editor de gráficos vetoriais de qualidade profissional para Linux, Mac OS X e Windows. É gratuito e de código aberto. Escolhemos o Inkscape por ser mais rápido para o desenvolvimento de vetores e auto-vetorização de imagens para o site. Saiba mais sobre o Inkscape aqui.

    • GIT

      Git é um gratuito sistema de controle de versão e hospedagem de conteúdo desenvolvido para hospedar de tudo, desde um projeto pequeno até projetos enormes que exigem rapidez e eficiência. O GIT hospeda o nosso site. Saiba mais sobre o GIT aqui.

    • HTML5/CSS3/JavaScript

      HTML5, CSS3 e JavaScript: estas foram as linguagens de formatação e programação que escolhemos para desenvolver nosso site. Estas tecnologias nos ajudam a tornar este site adaptável a qualquer dispositivo, a qualquer velocidade de conexão. Estas tecnologias são compatíveis com o WebKit, a web browser engine usada pelo Google Chrome, Firefox, Safari, App Store, e muitos outros aplicativos do Linux, OS X, iOS e Android.

  • O código deste site está disponível para todXs gratuitamente

Você pode baixar todo o código deste site na página do nosso projeto no GitHub, ele está disponível sob uma licença GNU Affero GPL 3.0.

  • Design do site

    Este site nasceu com o proposito de espalhar pela internet o nosso projeto ainda jovem, mas que pretende estar presente em todas as ruas desse país. Vamos levar arte para combater as injustiças sociais. E você pode se transformar num agente da mudança. Com esse objetivo nós desenvolvemos este espaço que conta com um design simples, responsivo, acessível e moderno visando a agilidade da informação para que você tenha acesso ao nosso conteúdo. O design deste site foi concebido por Flávio Vivório Carneiro, formado como técnico em informática pelo Instituto Federal Fluminense e que atualmente estuda no curso superior em design gráfico na mesma instituição. O design deste site é baseado no trabalho da web designer Mary Lou do blog de design Codrops.

    Acesse o nosso manual da identidade visual, clique aqui.

  • Design dos lambes

    Os lambes são desenvolvidos pelo Coletivo LAMBE QUE EU GOSTO, formado por estudantes de design gráfico e arquitetura e urbanismo do Instituto Federal Fluminense (IFF). O objetivo da LAMBE QUE EU GOSTO é levar arte e ativismo para as ruas de todo lugar.
